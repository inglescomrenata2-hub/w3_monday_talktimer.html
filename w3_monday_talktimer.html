Quick rule reminder

◆ Will vs Going to — when to use each

Will
Predictions — things you think will happen:
"I think it'll be a busy year."
Instant decisions — decided right now:
"I'll call her back."
Offers & promises:
"I'll help you with that."
Going to
Plans already decided:
"I'm going to start a new project next month."
Predictions with evidence:
"Look at that schedule — it's going to be crazy."
Intentions:
"I'm going to focus on speaking this year."
"I'll answer that" (spontaneous) vs "I'm going to answer all emails before lunch" (already planned)
Topics

◆ Speaking Tips

Antes de começar, pense: este plano já estava decidido? → going to. Decidi agora? → will.
Se travar, use "I think…" + will ou "I'm planning to…" + going to para comprar tempo.
Não pare se errar — native speakers às vezes usam will e going to de forma intercambiável. O contexto é o que importa.
Grave o áudio — ouça e conte quantas vezes usou cada forma.
